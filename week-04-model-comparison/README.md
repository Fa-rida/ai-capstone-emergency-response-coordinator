# Week 4: Model Comparison
Tested 4 AI models on 5 cybersecurity text samples to evaluate their suitability
for the Task Tracking & Assignment component of our Emergency Response Coordinator project.
## Models Tested
- HF Sentiment (distilbert-sst-2)
- HF Zero-Shot (bart-large-mnli)
- HF NER (bert-large-NER)
- Groq Llama 3 8B
## Finding
Recommended Groq Llama 3 8B for the Task Tracking & Assignment because it provided clear severity classifications and explanations for each alert, making it more useful than the Hugging Face models for cybersecurity analysis.
See `report.md` for full analysis.
